# Booking Scheduler App

A web application designed to streamline scheduling workflows and reduce manual coordination in logistics operations.

## Features
- Submit booking requests
- Admin dashboard to manage bookings
- Prevent double booking
- Improved scheduling visibility

## Tech Stack
- Python (Flask)
- HTML/CSS
- SQLite

## Business Problem
Manual scheduling processes created inefficiencies, delays, and coordination overhead. This application centralizes booking management and improves operational efficiency.

## Impact
- Reduced manual coordination effort
- Improved visibility into booking activity
- Streamlined scheduling workflows

## Future Improvements
- User authentication
- Calendar integration
- UI enhancements
